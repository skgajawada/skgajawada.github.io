🖼️ Adding Media Files

Profile Photo:
1. Prepare a photo (400x400px recommended)
2. Place in: assets/images/profile.jpg
3. Or update path in portfolio-data.json
Certificates:
1. Create folder: assets/certificates/
2. Place PDFs: STC_001.pdf, FDP_018.pdf, etc.
3. Place JPGs: STC_001.jpg, FDP_018.jpg, etc.
4. Update filenames in certificates.json

🎨 Customization

Change Colors:
Edit assets/css/style.css root variables:
