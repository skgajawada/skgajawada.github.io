Academic Projects

A study on the effect of elastic-plastic material properties of metals on hardness values using finite element analysis

The object of this project is how the elastic-plastic material properties of metals on hardness values varies from theoretical to experimental and optimization of these hardness values using Finite Element Analysis

Finite Element Analysis Hardness Properties

Study of internal combustion engines & their components

The main aim of this project is to introduce the fundamentals of internal Combustion Engines & their components. The report traces the mechanism of operation of Internal Combustion Engines and its different kinds based on the operation mechanism, specially the two stroke and four stroke engines.

IC Engines Mechanism Studies

Hydraulic Robotic Arm for pick and place operation

The overall purpose of this project was to understand how Hydraulics works. The basic idea of a hydraulic system is that the force applied to one point is transmitted to another using fluid. The basic concept used behind the operation is PASCALS LAW.
